# Requisitos Funcionales - SIGDEF

## 1. Módulo Institucional (Federación y Clubes)
- **RF-01**: El sistema permitirá el registro de Clubes afiliados con su información legal y de contacto.
- **RF-02**: El sistema permitirá la gestión de autoridades de la federación y su visualización jerárquica.
- **RF-03**: El sistema permitirá a la federación habilitar o deshabilitar clubes según su estado administrativo.

## 2. Módulo de Atletas y Documentación
- **RF-04**: El sistema permitirá la carga masiva y unitaria de atletas asociados a un club.
- **RF-05**: El sistema validará la edad del atleta para exigir el registro de un Tutor Legal si es menor de 18 años.
- **RF-06**: El sistema permitirá subir archivos multimedia (DNI, Apto Médico) y marcar su estado (Pendiente, Aprobado, Vencido).
- **RF-07**: El sistema alertará visualmente sobre aptos médicos vencidos o por vencer.

## 3. Módulo de Competencias (Eventos)
- **RF-08**: El sistema permitirá la creación de Eventos con múltiples pruebas configurables (Bote, Categoría, Distancia).
- **RF-09**: El sistema permitirá a los clubes inscribir a sus atletas en pruebas para las que cumplan con la categoría y sexo requeridos.
- **RF-10**: El sistema generará el listado de inscriptos (Start List) para cada regata.

## 4. Módulo Financiero
- **RF-11**: El sistema integrará Mercado Pago para el cobro de cánones de inscripción y seguros.
- **RF-12**: El sistema actualizará el estado de pago del atleta automáticamente al recibir la confirmación de la pasarela.
- **RF-13**: El sistema permitirá la carga manual de pagos (Transferencia/Efectivo) por parte de la federación.

## 5. Módulo Mobile
- **RF-14**: La aplicación móvil permitirá a los delegados e inscritos ver el cronograma de regatas en tiempo real.
- **RF-15**: La aplicación móvil proporcionará acceso rápido a la credencial digital del atleta (Código QR).
