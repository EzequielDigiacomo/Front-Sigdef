# Documentación Técnica SIGDEF 2026

Bienvenido a la carpeta de documentación técnica de **SIGDEF** (Sistema de Gestión Deportiva Federativa). Esta documentación está diseñada desde la perspectiva de un desarrollador de software para facilitar el entendimiento, mantenimiento y escalamiento del ecosistema.

3.  **[03-REQUISITOS-FUNCIONALES.md](./03-REQUISITOS-FUNCIONALES.md)**: Listado de capacidades del sistema.

5.  **[06-MODELO-DATOS.md](./06-MODELO-DATOS.md)**: Entidades, relaciones (ERD) y diccionarios de datos.
6.  **[07-CASOS-DE-USO.md](./07-CASOS-DE-USO.md)**: Interacciones por actor (Federación, Club, Atleta).

8.  **[11-GUIA-DESARROLLO.md](./11-GUIA-DESARROLLO.md)**: Setup local, comandos útiles y debugging en **Android Studio**.

## 🚀 Guía de Inicio Rápido para Devs

1.  Revisa el **[Resumen Ejecutivo](./01-RESUMEN-EJECUTIVO.md)** para entender el "Big Picture".
2.  Sigue la **[Guía de Desarrollo](./11-GUIA-DESARROLLO.md)** para levantar los entornos Front y Back.
3.  Si vas a trabajar en la App Móvil, asegúrate de sincronizar Capacitor: `npx cap sync`.
